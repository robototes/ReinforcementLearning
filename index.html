Reinforcement Q-Learning

A FIRST robot that learns how to operate itself, written in Java. Designed to run on a National Instruments cRIO-FRC II.

Files

QEstimator.java is the neural network used to estimate Q-Values for the learner. QLearner.java is the actual class that learns how to act. QNetworkSaver.java is used to save weights and biases of the neural network.
